# Music Folder API

API NestJS para el proyecto Music Folder.

## Despliegue en Render o Railway

1. Sube este repositorio a GitHub.
2. Crea un nuevo servicio en Render o Railway.
3. Selecciona la carpeta `apps/api` como directorio raíz del servicio.
4. Usa el comando de build:
   - `npm install`
   - `npm run build`
5. Usa el comando de inicio:
   - `npm run start`

El servicio expondrá la API en el puerto definido por la variable de entorno `PORT`.
