# Music Folder Web

Frontend React + TypeScript para el proyecto Music Folder.

## Despliegue en Vercel

1. Sube este repositorio a GitHub.
2. En Vercel, crea un nuevo proyecto.
3. Selecciona la carpeta `apps/web` como Root Directory.
4. Usa estos valores:
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. Haz Deploy.

## Desarrollo local

```bash
npm install
npm run dev
```
